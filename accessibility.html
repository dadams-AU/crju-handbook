Accessibility Statement

Last updated: July 11, 2026

Commitment to Accessibility

California State University, Fullerton (CSUF) is committed to ensuring digital accessibility for all users, including those with disabilities. The CRJU Undergraduate Advising Handbook is designed to be accessible to everyone, regardless of their technology or abilities.

Standards and Compliance

This website aims to conform to the Web Content Accessibility Guidelines (WCAG) 2.2 Level AA. We continuously review and improve accessibility features.

Conformance Level

WCAG 2.2 Level AA — This means our content is accessible to most users with disabilities, including those who use:

  • Screen readers
  • Screen magnification software
  • Keyboard-only navigation
  • Voice recognition software

Accessibility Testing

We regularly test this website with assistive technologies to ensure compatibility.

Assistive Technologies Tested

  • Screen readers: NVDA, JAWS, VoiceOver (Mac/iOS), TalkBack (Android)
  • Keyboard navigation: Tab key, arrow keys, skip links
  • Screen magnification: ZoomText, browser zoom up to 200%
  • Voice control: Dragon NaturallySpeaking, Voice Control (iOS)

Automated Testing

We use automated tools including:

  • WAVE (Web Accessibility Evaluation Tool)
  • Lighthouse Accessibility Audit
  • axe DevTools

Accessibility Features

  • Keyboard navigability: All interactive elements are accessible via keyboard
  • Screen reader compatibility: Proper ARIA labels, roles, and semantic HTML
  • High-contrast text: Minimum 4.5:1 contrast ratio for all text
  • Responsive design: Works on all screen sizes and orientations
  • Focus indicators: Visible focus states for all interactive elements
  • Alt text: Descriptive alternative text for all images
  • Heading structure: Logical heading hierarchy for easy navigation

Known Limitations

Despite our best efforts, some content may not be fully accessible. We are working to address these issues:

  • Some PDF documents may not be fully accessible (working on remediation)
  • Third-party embedded content may have varying accessibility
