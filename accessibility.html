Accessibility Statement

Last updated: July 11, 2026

Commitment to Accessibility

California State University, Fullerton (CSUF) is committed to ensuring digital accessibility for all users, including those with disabilities. The CRJU Undergraduate Advising Handbook is designed to be accessible to everyone, regardless of their technology or abilities.

Standards and Compliance

This website aims to conform to the Web Content Accessibility Guidelines (WCAG) 2.2 (opens in new tab) Level AA. We continuously review and improve accessibility features.

Conformance Level

WCAG 2.2 Level AA — This means our content is accessible to most users with disabilities, including those who use:

  • Screen readers
  • Screen magnification software
  • Keyboard-only navigation
  • Voice recognition software

Accessibility Testing

This website was audited against WCAG 2.2 Level AA in July 2026. Testing performed to date includes:

  • Automated scanning: axe-core run on every page, in both the light and dark color themes, with all interactive tools exercised
  • Keyboard navigation: scripted keyboard-only testing of every interactive element — tab order, skip link, focus visibility, Escape handling, and absence of keyboard traps
  • Color contrast: contrast ratios computed from rendered colors for both themes
  • Zoom and reflow: layout verified at 320px viewport width (equivalent to 400% zoom)
  • Reduced motion: animations and smooth scrolling respect the prefers-reduced-motion setting

Accessibility Features

  • Keyboard navigability: All interactive elements are accessible via keyboard
  • Semantic structure: Landmark regions, logical heading hierarchy, native HTML controls, and ARIA state attributes on custom widgets
  • Status announcements: Progress updates and dynamically revealed content are announced to screen readers via live regions
  • Color contrast: Text and UI indicators meet WCAG AA contrast minimums in both light and dark themes
  • Responsive design: Works on all screen sizes and orientations without horizontal scrolling
  • Focus indicators: Visible, theme-aware focus states for all interactive elements
  • Alt text: Descriptive alternative text for all images
  • New-tab warnings: Links that open in a new tab say so to assistive technology

Known Limitations

Despite our best efforts, some content may not be fully accessible. We are working to address these issues:

  • Live screen reader testing (NVDA, VoiceOver) has not yet been completed; the site follows ARIA best practices and a live assistive-technology pass is planned
  • The downloadable PDF version of the handbook may not be fully accessible (working on remediation)
  • Third-party embedded content may have varying accessibility
